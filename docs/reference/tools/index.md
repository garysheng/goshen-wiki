# Tools

*One page per tool. What it is, what it costs, when to use it, and when not.*

---

Every tool page in this wiki follows the same shape:

- **What it is** — one paragraph, plain.
- **What it costs** — pricing tiers, the honest version.
- **When to use it (and when not)** — paired bullets. The "not" half is load-bearing.
- **Setup pointer** — link to official docs. We do not duplicate installation steps.

See `templates/tool.mdx` at the repo root for the copy-and-rename scaffold.
